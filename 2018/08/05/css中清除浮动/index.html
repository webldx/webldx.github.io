第四种

​ 给父级元素添加一个属性 overflow:hidden

​ 这写法的原理就是必须定义width或zoom:1，同时不能定义height，使用overflow:hidden时，浏览器会自动检查浮动区域的高度

  • 优点：简单，代码少，浏览器支持好
  • 缺点：内部宽高超过父级div时，会出现滚动条。
  • 建议：不推荐使用，如果你需要出现滚动条或者确保你的代码不会出现滚动条就使用吧。
  • 评分：★★☆☆☆
